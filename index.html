    Peer Mesh #

    A wrapper for PeerJS that makes reactive peer-to-peer connections easy using Zustand.

    Installation #

    npm install @galileocap/peer-mesh

    Quick Start #

    Basic use #

    import { PeerStore } from '@galileocap/peer-mesh';
const usePeerStore = new PeerStore();
